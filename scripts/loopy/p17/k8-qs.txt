Quicksort is a highly efficient, "Divide and Conquer" algorithm used to sort arrays. It works by selecting a "pivot" element, partitioning the array around that pivot, and recursively sorting the resulting sub-arrays.

Here is a detailed step-by-step explanation of how it works, followed by a Python implementation.

### The Core Concept: Divide and Conquer
Quicksort operates in three main phases:
1.  **Divide:** Pick a pivot and partition the array.
2.  **Conquer:** Recursively apply the algorithm to the sub-arrays.
3.  **Combine:** The array is sorted in-place (no merging step is required).
