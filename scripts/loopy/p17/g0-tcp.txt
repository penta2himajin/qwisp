        *   TCP needs to share bandwidth fairly and avoid dropping packets.
        *   Key variable: `cwnd` (Congestion Window). It dictates how many bytes can be in flight.

    *   **Slow Start:**
        *   *Why?* To find the available bandwidth quickly without probing too aggressively.
        *   *Mechanism:* Exponential growth.
        *   *Start:* `cwnd` starts at 1 segment (or 10 bytes in modern implementations).
        *   *Action:* When an ACK arrives, `cwnd` increases by 1 segment (MSS).
        *   *Result:* 1 -> 2 -> 4 -> 8 -> 16... (doubling per RTT).
        *   *Termination:* When `cwnd` reaches `ssthresh` (Slow Start Threshold), it switches to Congestion Avoidance.

    *   **Congestion Avoidance (Linear Growth):**
        *   *Why?* To be "gentle" once we know the rough capacity.
        *   *Mechanism:* Additive Increase.
        *   *Action:* Increase `cwnd` by 1 segment per RTT (per ACK).
        *   *Result:* 16 -> 17 -> 18... (linear).
        *   *Termination:* If a loss is detected, `cwnd` drops to 1 (or half), and we go back to Slow Start.

    *   **Fast Retransmit (The "3x" Rule):**
        *   *Problem:* Standard timeout (RTO) is too slow (seconds).
        *   *Solution:* Detect loss faster.
        *   *Mechanism:* Duplex ACKs (Duplicate ACKs).
        *   *Rule:* If 3 duplicate ACKs arrive for the same sequence number, assume the packet was lost (not just reordered).
        *   *Action:* Retransmit the lost segment immediately.
        *   *State Change:* Enters "Fast Recovery" state.

    *   **Fast Recovery (The "Recovery" Phase):**
        *   *Problem:* Standard recovery resets `cwnd` to 1 (Slow Start). This is inefficient.
        *   *Solution:* Keep `cwnd` high but "safe".
        *   *Mechanism:*
        *   *State:* Fast Recovery.
        *   *Action:* When sending new data (piggybacked on ACKs), increase `cwnd` by 1 segment per ACK (like Congestion Avoidance).
        *   *Termination:* When the lost segment is retransmitted and acknowledged, exit Fast Recovery.
        *   *Reset:* Set `ssthresh` to `cwnd / 2` (halving the window). Restart Slow Start.
